# cf-assess

CrossFit performance assessment tool for CrossFit Vic City.

Single-file HTML app — no backend, no build step, no dependencies. Open `index.html` in any browser. All data stored locally in `localStorage`.

**Features**
- Up to 10 athlete profiles with PIN authentication
- 9 benchmark workouts (Gymnastics, Olympic Lifts, Conditioning) with Rx / Scaled / Modified tracking
- A/B/C/D grading against real performance standards, with Garmin HR input
- Injury/limitation flags that restrict benchmarks and surface modification notes
- Self-assessment questionnaire (18 questions, 3 domains) with domain scoring and accessory prescriptions
- Grade-specific accessory work drawers per benchmark

**Live app:** https://mattziegler12-prog.github.io/cf-assess/
